Linguagem de Programação

A disciplina Linguagem de Programação - LP aborda conceitos, paradigmas, elementos de organização e gerenciamento de linguagem, além de possibilitar a aplicação dos estudos sobre algoritmos a uma linguagem de programação estruturada. A partir de estudos teóricos e práticos, o/a aluno/a poderá desenvolver programas para a solução de problemas em uma linguagem de programação de alto nível, fazer verificações e correção de bugs, além de implementar programas modularizados.
